# Карта MCP-инструментов

## Инфраструктура (разработано в проекте)

Всё работает поверх собственного Docker-стека.

| Компонент | Технология | Роль | Порт |
|---|---|---|---|
| **Parser** | Python / FastAPI | Читает XML-выгрузку конфигурации 1С, отдаёт объекты по REST | 8001 |
| **Embeddings** | `sergeyzh/BERTA` (768d) | Векторизует описания и BSL-код; dense + sparse (BM25) | 5050 |
| **Qdrant** | v1.13.2 | Векторная БД: коллекции `metadata_1c` и `code_1c` | 6333 |
| **Loader** | Python / Streamlit | UI индексации: «Метаданные» и «BSL-код» | 8501 |
| **MCP Server** | TypeScript / Node.js | HTTP MCP-сервер (Streamable HTTP + SSE) | 8000 |

```bash
docker compose up -d
# Loader UI  → http://localhost:8501
# MCP        → http://localhost:8000/mcp
```

## Инструменты для разработчика 1С

### Наши (встроены в MCP Server)

| Инструмент | Описание |
|---|---|
| `1c_metadata_search` | Поиск объектов конфигурации по имени или синониму |
| `1c_metadata_details` | Полная структура объекта: реквизиты, ТЧ, движения в регистры |
| `1c_metadata_types` | Статистика по типам объектов в конфигурации |
| `1c_code_search` | Гибридный поиск по BSL-коду: процедуры, функции (коллекция `code_1c`) |

### Внешние (comol, запускаются отдельно)

| Инструмент | Docker-образ | Описание | Приоритет |
|---|---|---|---|
| **HelpSearchServer** | `comol/1c_help_mcp` :8003 | Справка по платформе вашей версии | 🔴 Высокий |
| **Graph_metadata_search** | comol | Графовый поиск по метаданным и связям объектов | 🔴 Высокий |
| **CodeMetadataSearchServer** | `comol/1c_code_metadata_mcp` :8000 | Поиск паттернов по коду + справка конфигурации | 🟡 Средний |
| **SSLSearchServer** | `comol/mcp_ssl_server` :8008 | Справка по БСП нужной версии | 🟡 Средний |
| **TemplatesSearchServer** | `comol/template-search-mcp` :8004 | Поиск шаблонов кода + добавление своих | 🟡 Средний |
| **SyntaxCheckServer** | `comol/1c_syntaxcheck_mcp` :8002 | Проверка синтаксиса BSL через BSL Language Server | 🟡 Средний |
| **1CCodeChecker** | `comol/1c-code-checker` :8007 | Проверка кода через 1С:Напарника (нужен токен партнёра) | 🟢 Низкий |
| **FormsServer** | `comol/1c_forms` :8011 | Контекст для генерации форм | 🟢 Низкий |
